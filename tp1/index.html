1. ¿Qué es un lenguaje de marcado? ¿Cuál es su utilidad? ¿Qué es un tag? ¿Qué es un atributo?

Un lenguaje de marcado es un lenguaje donde el texto se encuentra etiquetado de cierta manera. La utilidad de esto es que se pueda leer el texto e interpretar las etiquetas y lo que está dentro de las mismas, realizando distintas acciones o simplemente identificando el contenido.

Los tags o etiquetas son los elementos que proveen esta identificación; por ejemplo, en HTML, un texto que se encuentra entre las etiquetas <p> y </p> es identificado como texto de párrafo. Los atributos son propiedades definidas dentro de un tag para controlar el comportamiento del mismo; por ejemplo, el atributo href especifica la URL de la página a la cual apunta un enlace, como puede ser el del tag <a> en HTML.


2. ¿Cuál es la utilidad de HTML? ¿Qué conjunto mínimo de tags debe contener un documento elaborado en este lenguaje? Describa brevemente su utilidad.

HTML (Hypertext Markup Language) sirve para describir semánticamente un documento, actualmente en su mayoría páginas web.

Los tags mínimos necesarios definidos por la W3C son los siguientes:

  • <html> indica el comienzo del documento (raíz).

  • <head> contiene todos los elementos de cabecera, como pueden ser el título del documento, scripts, estilos, etc. Los tags que pueden ser definidos dentro de <head> son <title>, <style>, <base>, <link>, <meta>, <script> y <noscript>.

  • <title> define el título del documento.

  • <body> define el cuerpo del documento. Dentro del elemento <body> se encuentran los contenidos del documento HTML, incluyendo texto, imágenes, listas, etc.

Aunque no es un tag HTML, todo documento HTML debe tener la declaración <!DOCTYPE html> al principio para que el navegador sepa que se trata de un documento HTML.


3. ¿Cuál es la utilidad e importancia de los enlaces o links entre páginas? ¿Qué significa hipertexto? ¿Un link solo puede apuntar a otra página? ¿Qué importancia tiene esto último?

Los enlaces o hipervínculos permiten a los usuarios navegar de manera más rápida y cómoda, ya que no tienen que manualmente escribir la URL del recurso al cual quieren acceder, sino que las páginas se interconectan unas con otras para formar sitios complejos o apuntar a otros sitios.

El hipertexto es texto con referencias (hipervínculos) a otros recursos a los cuales se puede acceder. Un hipervínculo no necesariamente apunte a otra página, sino que puede hacer referencia a otro texto dentro de la misma página (caso más común de los anchors en HTML). Esto permite al usuario navegar en la página sin necesidad de hacer una nueva petición al servidor cada vez que se accede a un enlace.
