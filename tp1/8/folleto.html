The Evolution Of <Web Design>

  1. 1990

    HTML

    The First markup language was launched to code basic building blocks of a website.

  2. 1991

    1st Website Created

    Tim Berner's Lee - Also known as TimBL is a British Computer Scientist & Creator of WWW

  3. 1992

    Table Based Web

    Table designs gave more options for creating websites but were extremely complex.

  4. 1994

    W3C formed

  5. 1996

    Flash & Javascript

    Flash & Javascript introduced for a more interactive web.

  6. 1998

    • CSS Design

      Faster loading CSS websites come into being & cost 10x the table design.

    • PHP websites

      PHP3 websites pave the way for Dynamic Websites.

  7. 2003

    CSS 3 & Web 2.0

    CSS3 Introduces new Features & function. Web2.0 introduces age of user based information sites.

  8. 2008

    Mobile Web

    Mobile web takes over, more websites now develop Mobile version of sites & Apps.

  9. 2010

    HTML 5

    HTML5 websites are accesible across different devices & provide more enrich embedded media experience.
